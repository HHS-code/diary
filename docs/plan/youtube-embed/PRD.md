# PRD — 유튜브 임베드 카드 (youtube-embed)

## 배경 / 왜

지금까지 다이어리 캔버스에는 이미지, 텍스트, 스티커, GIF, 자유 그리기 오브젝트를 올릴 수 있다. 하지만 "오늘 본 영상"을 기록하려면 텍스트로 링크만 적어두거나 별도로 캡처해야 한다. 유튜브 링크를 캔버스에 붙여넣으면 바로 재생 가능한 카드로 표시되게 해, 일기 기록의 자연스러운 확장으로 만든다.

## 목표

- 유튜브 링크를 캔버스에 붙여넣으면(Ctrl+V) 자동으로 감지해 썸네일 카드 오브젝트를 생성한다.
- 카드를 클릭하면 캔버스 안에서 바로 영상이 재생된다(새 탭/모달 없음).
- 저장(JSON export/import) 후에도 카드가 유지된다.

## 대상 사용자

기존과 동일. 혼자 쓰는 개인 다이어리 사용자.

## In-scope

### 1. 유튜브 링크 감지 및 카드 생성

- `useCanvasKeyboardShortcuts`의 기존 붙여넣기(`handlePaste`) 흐름을 확장한다.
- Ctrl+V 시 클립보드 텍스트가 유튜브 URL 패턴(`youtube.com/watch?v=`, `youtu.be/`, `youtube.com/shorts/` 등)과 일치하면, videoId를 추출해 카드 오브젝트를 캔버스에 추가한다.
- 기존 이미지 붙여넣기 우선순위(네이티브 파일 → 클립보드 이미지 → 오브젝트 붙여넣기)에 유튜브 URL 감지를 추가한다. 정확한 우선순위는 architecture에서 결정.

### 2. 카드 모양

- 유튜브 표준 16:9 비율의 썸네일 카드. 썸네일 위에 재생 버튼(▶) 아이콘이 오버레이된 형태.
- 썸네일 이미지는 `img.youtube.com/vi/{videoId}/...` 외부 URL을 그대로 사용한다(온라인 전제, 오프라인이면 깨진 이미지 허용).
- 다른 캔버스 오브젝트처럼 이동·16:9 비율 유지 크기조절·삭제 가능(canvas-object-toolbar 재사용).

### 3. 인라인 재생

- 카드를 클릭하면 카드와 같은 위치·크기에 유튜브 iframe이 오버레이되어 캔버스 안에서 바로 재생된다.
- 재생 중에는 카드(오버레이) 이동만 가능하다. 회전·크기조절은 잠근다.
- 캔버스가 확대/축소/팬 되거나 카드가 이동하면 iframe 위치도 함께 동기화된다.
- 재생을 멈추는 방법(카드 바깥 클릭 등)은 architecture에서 결정.

### 4. 저장/불러오기

- JSON export에는 videoId만 저장한다. 불러올 때 썸네일 URL을 videoId로부터 재조립한다.
- PNG export(모아보기 갤러리 등)에는 재생 상태와 무관하게 항상 썸네일 이미지로 찍힌다(animated-gif-support의 정지 이미지 대체 패턴과 동일한 방향).

## Out-of-scope (명시)

- 유튜브 외 링크(트위터/인스타/일반 웹페이지) 카드 — `social-embed` 후속 phase 후보로 분리.
- JSON export를 HTML export로 전환하는 것 — `html-export` 후속 phase 후보로 분리. 이번 phase는 기존 JSON export 체계를 그대로 유지.
- "유튜브 추가" 전용 버튼/입력창 UI — 이번엔 붙여넣기 자동 감지만 지원.
- 오프라인 대응(썸네일 캐싱, 로컬 저장) — 온라인 전제.
- 재생 중 회전/크기조절 — 이동만 허용.
- 재생 속도, 반복재생 등 플레이어 옵션 커스터마이징 — 유튜브 기본 플레이어 동작 그대로.

## 성공 기준

- 유튜브 링크(watch URL, 단축 URL, shorts URL)를 캔버스에 붙여넣으면 썸네일 카드가 생성된다.
- 카드를 클릭하면 캔버스 안에서 영상이 재생된다.
- 재생 중 카드를 이동할 수 있고, iframe이 카드 위치를 따라간다.
- 저장 후 다시 불러오면 카드가 동일하게 복원된다(썸네일도 다시 뜸).
- 캔버스를 PNG로 export하면 재생 중이었더라도 썸네일 이미지로 찍힌다.
- 기존 오브젝트 툴바(복사/삭제/레이어순서, Ctrl+C/V, Delete)가 유튜브 카드에도 동일하게 동작한다.

## 진행 방식

1. 문서(PRD/architecture) 합의 → 실행 승인
2. URL 감지 유틸 → 커스텀 Fabric 오브젝트(카드) → 붙여넣기 흐름 연결 → iframe 오버레이 재생 → JSON 저장/불러오기 → PNG export 처리 → 완료 시 사용자에게 결과 설명
