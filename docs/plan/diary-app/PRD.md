# PRD — 자유 커스터마이징 다이어리 웹앱

## 배경 / 왜

서버 없이 완결되는, 자유롭게 꾸미는 개인 다이어리 웹앱을 만든다. 텀블벅 "딩글" 프로젝트를 레퍼런스로 검토했으나, 해당 프로젝트는 "링크 스크랩 + 위젯 대시보드" 컨셉으로 본 프로젝트의 방향(자유 배치 캔버스형 다꾸)과 결이 달라 UI 세부사항은 참고하지 않는다.

## 목표

- 날짜별로 이모지 스티커·이미지·사진·손글씨 텍스트를 자유롭게 배치해 꾸미는 다이어리를 만든다.
- 데이터는 기기 안에만 존재하며(localStorage), JSON export/import로 기기 간 이동이 가능하다.
- 꾸민 페이지를 PNG로 내보내 공유할 수 있다.
- (2차) 영화를 검색해 정보를 자동으로 불러오고 별점을 매기며, 같은 자유 캔버스로 꾸밀 수 있다.

## 대상 사용자

혼자 쓰는 개인 다이어리 사용자. 로그인/계정 없음, 기기 간 실시간 동기화 없음.

## In-scope

### 1순위 — 자유 커스터마이징 다이어리

- React + Vite 프론트엔드, 서버/백엔드 없음 (전부 클라이언트 동작)
- Fabric.js 기반 캔버스 에디터
  - 오브젝트 자유 배치: 드래그 이동, 크기조절, 회전
- 기본 이모지 스티커 팔레트 (고정 목록 중 선택해 캔버스에 추가)
- 사용자 이미지 업로드 → 캔버스에 스티커처럼 배치
- 손글씨 느낌 폰트의 텍스트 메모 추가 (한글 + 영어 지원)
- 사진 업로드 → 캔버스에 배치
- 날짜별 페이지 관리
  - 캘린더 UI로 날짜 선택
  - 같은 날짜 안에 "다이어리" 탭과 "영화리뷰" 탭 공존 (탭 전환)
- 저장: localStorage에 JSON 자동 저장, 새로고침해도 유지
- 내보내기 / 불러오기: JSON 파일 다운로드 · 업로드 (기기 이동용)
- PNG 이미지로 내보내기 (공유용)

### 2순위 — 영화 리뷰 (TMDB API 연동, 1순위 완료 후 진행)

- 영화 검색 → 포스터 · 제목 · 감독/배우 · 줄거리 자동 조회 (TMDB API)
- 5점 별점
- 같은 날짜의 "영화리뷰" 탭 안에 영화 정보 카드 + 자유 캔버스(1순위와 동일한 에디터)가 공존하는 레이아웃
- TMDB API 키는 `.env` 파일로 분리, `.gitignore`에 추가 (하드코딩 금지, 커밋 방지)

## Out-of-scope (명시)

- 서버/백엔드, 로그인/계정, 다중 기기 실시간 동기화 — JSON export/import가 유일한 이동 수단
- 협업 / 공유 실시간 편집 기능
- 네이티브 모바일 앱 (웹앱만)
- 위젯형 카드 일체 (NAME 프로필, MONTHLY GOALS, EVENT D-day, BUCKET LIST, 캘린더 위젯) — 다음 기획으로 이월
- 유튜브 등 링크 자동 임베드 카드 — 다음 기획으로 이월
- Letterboxd 스타일 "좋아하는 영화 4선" / "최근 본 영화 목록" 위젯 — 다음 기획으로 이월

## 성공 기준

- **1순위**: 캘린더에서 날짜를 선택하면 해당 날짜의 다이어리 페이지로 이동하고, 이모지·업로드 이미지·사진·손글씨 텍스트를 캔버스에 자유롭게 추가·이동·크기조절·회전할 수 있다. 새로고침해도 내용이 유지되고, JSON 다운로드/업로드로 다른 기기에 옮길 수 있으며, PNG로 내보낼 수 있다.
- **2순위**: 같은 날짜의 "영화리뷰" 탭에서 영화를 검색해 포스터/제목/감독·배우/줄거리를 자동으로 불러오고 5점 별점을 매길 수 있으며, 같은 탭 안에서 자유 캔버스로 추가 꾸미기가 가능하다.

## 진행 방식

1. 문서(PRD/architecture) 합의 → 실행 승인
2. 1순위 기능 구현 → 완료 시 사용자에게 결과 설명
3. 2순위(TMDB 연동) 진행 → 완료 시 사용자에게 결과 설명
