# Step 3: 이미지 업로드와 사각형 크롭

## 이 단계를 왜 했나?

이전 단계(step 2)에서 스티커 스튜디오는 그림을 그릴 수 있는 빈 투명 캔버스만 있었다. 그런데 스티커는 직접 그린 그림뿐 아니라 "사진을 올려서 편집한 결과"로도 만들 수 있어야 한다(PRD 목표). 그러려면 먼저 사진을 캔버스에 올리는 방법과, 필요 없는 부분을 잘라내는(크롭) 방법이 있어야 한다. 이번 단계는 이 두 가지를 만들었다.

## 무엇을 만들었나?

1. **이미지 업로드 버튼** — 사이드바의 "이미지 업로드" 버튼을 누르면 파일 선택 창이 뜨고, 사진을 고르면 캔버스에 바로 올라간다. 스티커 캔버스(512×512픽셀)보다 큰 사진은 비율을 유지한 채 캔버스 안에 들어오도록 자동으로 줄어든다.
2. **사각형 크롭** — "자르기" 버튼을 누르면 크롭 모드로 들어간다. 캔버스 위에서 마우스를 드래그하면 점선 사각형이 그려지고, 이 사각형이 "남길 영역"이다. "크롭 적용"을 누르면 그 영역만 남고 나머지는 모두 사라진다(잘라낸 결과 하나만 새 그림으로 캔버스에 남는다). "취소"를 누르면 점선 사각형만 지워지고 원래 그림은 그대로 남는다.

이번 단계에서는 IndexedDB(브라우저 안에 파일을 저장하는 저장소)에 저장하는 동작은 하지 않는다. 업로드하거나 크롭한 이미지는 아직 "편집 중"인 임시 상태이고, 실제 저장은 이후 단계에서 만들 "완성" 버튼을 눌러야 일어난다(ADR-4 문서에서 이미 정한 "다른 이름으로 저장" 원칙과 관련).

## 데이터가 어떻게 흐르나?

**업로드**

```
사용자가 파일 선택
  → 브라우저가 그 파일을 가리키는 임시 주소(objectURL) 생성
  → 그 주소로 이미지를 읽어와 캔버스 오브젝트로 만듦
  → 캔버스(512x512)보다 크면 비율 유지하며 줄임
  → 캔버스 한가운데 배치
```

예시: 1024×768 사진을 올리면, 512(캔버스 폭) ÷ 1024(사진 폭) = 0.5배로 줄어서 512×384 크기로 캔버스에 들어온다. 원래 300×200처럼 캔버스보다 작은 사진은 줄이지 않고 그대로 가운데에 놓인다.

**크롭**

```
"자르기" 클릭 → 크롭 모드 시작 (다른 오브젝트를 실수로 선택/이동하지 않도록 임시로 막음)
  → 마우스 드래그: 시작점(10,20) ~ 끝점(110,220)
    → 점선 사각형: 왼쪽 10, 위 20, 너비 100, 높이 200
  → "크롭 적용" 클릭
    → 점선 사각형 안쪽만 실제 그림으로 다시 그림(잘라냄)
    → 캔버스에 있던 이전 그림들은 모두 지움
    → 잘라낸 결과 하나만 캔버스 가운데에 다시 놓음
```

"취소"를 누르면 점선 사각형만 지우고 위 과정은 실행하지 않는다.

## 쉽게 말하면

포토샵이나 그림판의 "사각형 선택 후 자르기"와 똑같은 동작이다. 종이 위에 사진을 붙이고, 필요한 부분에 네모를 그린 다음 가위로 오려내는 것과 같다 — 네모 밖은 버리고 네모 안만 새 종이(캔버스)에 다시 붙인다.

## 제대로 됐는지 어떻게 확인했나?

- 코드 검사(lint), 빌드(build), 자동 테스트(test) 모두 통과(테스트 29개 파일, 163개 테스트 전부 성공).
- 새로 만든 자동 테스트로 확인한 것:
  - 캔버스보다 작은 이미지는 원본 크기로, 큰 이미지는 비율을 유지하며 줄어들어 캔버스 안에 들어온다.
  - 업로드한 이미지는 아직 저장된 것이 아니므로 저장소 식별자(assetId)가 없다.
  - 크롭 영역을 드래그하면 점선 사각형이 정확한 위치·크기로 캔버스에 나타난다.
  - "취소"를 누르면 사각형만 사라지고 기존 그림은 그대로 남는다.
  - "크롭 적용"을 누르면 이전 그림들은 모두 사라지고 잘라낸 결과 하나만 남는다.
  - 영역을 드래그하지 않고 바로 "크롭 적용"을 누르면 아무 일도 일어나지 않는다(빈 크롭 방지).
- 실제 브라우저에서도 직접 확인: 스티커 스튜디오를 열어 이미지를 업로드하고, 일부 영역을 드래그로 지정한 뒤 "크롭 적용"을 눌러 그 영역만 남는 것을 화면으로 확인했다.

## 전체 그림에서 지금 어디?

```
[완료] 0. 캔버스 크기 조절 가능하게
[완료] 1. 스티커 저장 형식 준비
[완료] 2. 스티커 스튜디오 창 열기 + 그리기 도구
[완료] 3. 이미지 업로드 + 사각형 크롭   ← 지금 여기
[남음] 4. 올가미로 배경 제거(누끼따기)
[남음] 5. 흰색 테두리 추가
[남음] 6. 저장하고 다이어리에서 꺼내 쓰기
```

다음 단계는 "누끼따기"다. 지금까지는 이미지 전체 또는 사각형 영역 단위로만 다뤘는데, 다음 단계부터는 자유롭게 그린 곡선 안쪽만 남기고 바깥을 투명하게 만드는 작업이 필요하다. 이번 단계에서 만든 "캔버스 위에 임시 도형을 그렸다가 확정한다"는 방식(사각형 대신 자유곡선)을 그대로 이어서 쓸 수 있어서, 순서상 이번 단계 다음에 오는 것이 자연스럽다.

## 아직 안 된 것 (솔직하게)

- 업로드/크롭 결과는 아직 저장되지 않는다 — 브라우저를 새로고침하면 사라진다. 저장은 6단계("완성" 버튼)에서 만든다.
- 크롭은 사각형만 가능하다. 자유로운 모양으로 배경을 지우는 것은 다음 단계(누끼따기)의 몫이다.
- 업로드한 이미지 위에 그림을 덧그리는 것 자체는 이미 되지만(그리기 도구가 그대로 살아있음), 이번 단계에서 새로 테스트하지는 않았다 — 그리기 도구는 2단계에서 이미 검증된 기능이라 그대로 믿고 넘어갔다.
